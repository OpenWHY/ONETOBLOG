              # 指导思想

              一切工具的产生源于人类的懒，简单的工作重复做简直是一件让人崩溃的事

              PS: 并不是指所有的事情重复做都是没有意义！

              重点思考如何把日常工作做好、提高工作效率以及工作效能。

              # 发起原因

              前段时间由于工作安排，需要经常进行会务对接工作，经常性设计会场座次表，用 Word 或者 Excel 设计很土气，如果使用 Photoshop 之类的设计软件我的电脑非常卡而且个人设计能力也很有限，所以非常纠结，想开发一套排座系统，可以轻松输出漂亮的座次表！
